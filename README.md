# Курс «Алгоритмы и структуры данных»

## Описание
Проект содержит решения домашних заданий

## Установка
Для установки и запуска решений задач, необходим [NodeJS](https://nodejs.org/)
1. Клонируйте репозиторий.
2. Выполните команду `npm install`

## Использование
В проекте находится список папок (lesson-1, lesson-2 и т.д.) каждая из которых содержит решение домашнего задания.
Пример запуска решения первого задания:
```sh
$ npx tcs
$ node dist/lesson-1/index.js
```

## Тесты
Пример запуска тестов:
```sh
$ npm test
```
